    • lit-element은 virtual Dom과 다르지만, Dom을 완전히 직접 조작하는 것은 비효율적이다.

    사용하기

    prop

    converter

    type

    attribute (attr -> prop)

    reflect (prop -> attr)

    noAccessor

    noAccessor가 true면, prop이 getter와 setter 접근자에 의해 변경된다.
    noAccessor가 false면 getter와 setter가 사용되지 않는다.

    hasChanged

    true가 반환되면, update를 실행한다.
    false가 반환되면, 변화가 없다는 뜻이다.

    loop

    conditionals
